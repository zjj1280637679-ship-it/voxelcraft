## 0. 摘要

本表是《山根乡》**经济骨架的权威来源**:横向贯通三条获取路线(采集 / 狩猎 / 制造),纵向贯通五阶级(木→石→铁→黄金→钻石),逐格可溯源到白皮书。后续任何经济、平衡、制造、掉落设计都以本文为对账基准。

四件事在本文统一:
1. **材料阶级网格**(§1):每个阶级在三条路线上有什么材料、什么产地、什么韧性、什么采集门槛、什么装备。
2. **三套结算规则查表**(§2):战斗等级压制、采集门槛、韧性公式 —— 三者解耦、互不串味。
3. **掉落 / 食物 / 附性归一化**(§3):生物给玩家什么、食物怎么恢复、显现能附带什么,三者共用「原型锁定·显现自由」一条铁律。
4. **经济闭环与进度门**(§4):三路线 × 五阶级的触达矩阵、游玩层三门(工作台 × 图纸 × 材料)逐阶展开、冗余与断档的对抗性审查。

第 5 节汇总四节发现的全部漏洞与开放问题。

**核心数值锚(全文反复引用)**:
- 等级压制(WP §4.3):防−攻 ≤0=100% / 1=45% / 2=15% / 3=5% / 4=1%(**非零下限,有意为之**)。
- 采集门槛(WP §4.4 + 无免疫裁决):工具−目标 ≥0=正常 / −1=慢但保底掉落 / ≤−2=极慢减产但不归零(**无免疫,定性无倍率**)。
- 韧性(WP §5.2):最终韧性 = 基础韧性 × 形态系数 × 状态系数。

### 1.2.1 同阶采集器偏置表(同阶差异化唯一来源,WP §6.2)

> 采集工具已归一化为「采集器」(WP §6.1,取消镐/斧/锄等分裂工具)。**同阶采集器基础能力相同**,但可带**类别偏置**——这是同阶四种采集器之间唯一的经济差异来源,也是证伪「同阶采集器互为死冗余」的关键依据。偏置**不跨阶**(WP §6.2):偏置只调同阶内部的采集效率,不能把低阶采集器拉到能采高阶目标。采集类别见 WP §6.2(土系/石系/木系/植物系/骨系/特殊系)。

| 采集器 | 加成 | 减成 | 出处 |
|---|---|---|---|
| 矿工铲 | 石系 **+15%** | 植物系 **−5%** | WP §6.2 |
| 林工铲 | 木系 **+15%** | 石系 **−5%** | WP §6.2 |
| 泥工铲 | 土系 **+15%** | 骨系 **−5%** | WP §6.2 |
| 骨工铲 | 骨系 **+15%** | 植物系 **−5%** | WP §6.2 |

**结论**:同阶采集器靠 ±15% / −5% 的**类别偏置**区分,**非死冗余** —— 选哪把铲取决于主采类别(挖矿选矿工铲、伐木选林工铲、采骨选骨工铲)。但偏置幅度(±15%/−5%)是否足以构成「有意义的选择」而非可忽略的微差,白皮书未给经济权重判据(入 §5,Q25)。

## 2. 等级压制 · 采集门槛 · 韧性(三套规则查表)

> 本节为「战斗结算 / 采集判定 / 韧性计算」三套规则的可直接查阅版。阶级索引同 §1:木=0/石=1/铁=2/黄金=3/钻石=4,黑曜石=钢=黄金级=3。

### 2.0 三套规则的关系(互不串味)

```
       ┌─ 战斗:谁打谁 → 查【等级压制表】(防御等级 − 攻击等级)→ 最终伤害%
对象等级 ┼─ 采集:工具 vs 目标 → 查【采集门槛表】(工具等级 − 目标等级)→ 正常/很慢/无效
       └─ 物理:挖多久/多难破 → 查【韧性公式】(基础韧性 × 形态系数 × 状态系数)
```

三者解耦(WP §5.1 明确):**韧性只管「破坏/采集/抗爆」,不进武器伤害**;武器伤害只由「槽位 × 等级」决定(WP §8.1)。打怪查压制表、挖矿查采集表、「这块墙多硬」查韧性公式,三条线分开结算。

### 2.1 等级压制查表(战斗,WP §4.3)

**规则**:`差值 = 防御方等级 − 攻击方等级`,查表得最终伤害百分比(WP §4.3 标注为「推荐倍率」)。

| 防御等级 − 攻击等级 | 最终伤害 | 口语解释 |
|:---:|:---:|---|
| ≤ 0 | **100%** | 攻方等级 ≥ 防方,满伤 |
| 1 | **45%** | 高 1 级,打八折偏多 |
| 2 | **15%** | 高 2 级,刮痧 |
| 3 | **5%** | 高 3 级,几乎挠痒 |
| 4 | **1%** | 高 4 级,**仍非零**(见下「非零下限」) |

**关键性质 — 非零下限(锁定模型,有意为之)**:
- 表底是 **1%,不是 0%**。木级(0)打钻石级(4)差 4 级 → 1% 伤害,**仍能造成伤害**。
- 这意味着**理论上允许「低级屠龙」**——一群木剑玩家堆足够多次攻击,可以磨死 Boss。这是**有意设计**(锁定模型),不是漏洞:给低阶玩家保留「以量补质 / 多人围攻」的非零希望,避免「等级不够就完全打不动」的硬墙。
- 反向同理:高级打低级,差值 ≤ 0 恒 100%,**不会因「欺负小怪」被削**,高阶碾压低阶是干净满伤。

**单调性自洽**:差值 0→1→2→3→4 对应 100→45→15→5→1,严格单调递减且全程 > 0,无折返、无并列、无 0 值。

### 2.2 采集门槛查表(采集 / 挖掘,WP §4.4)

**规则**:`差值 = 工具等级 − 目标等级`,查表得采集效果。注意方向与压制表相反:**这里是「工具」减「目标」**。

| 工具等级 − 目标等级 | 效果 | 是否掉落 |
|:---:|:---|:---:|
| ≥ 0(工具 ≥ 目标) | **正常采集** | 正常掉落 |
| −1(工具低 1 级) | **慢,但可挖到** | **保底掉落** |
| ≤ −2(工具低 2 级及以上) | **极慢 + 减产** | 仍有产出,**永不归零** |

**采集亦「无免疫」(房主拍板,2026-06-13)**:与战斗同一原则——阶级差只**减速/减产**,**绝不归零**。先前草稿把「差 2 级」写成「采集硬墙/几乎无效」的非对称,**已按本裁决撤销**:差距越大越慢、越减产,但低阶工具**永远能慢慢挖到**高阶材料。防跳级靠**惩罚性低效**(差≥2 极慢,没人会拿它当主路),而非硬墙。

**消解白皮书内部矛盾(已决 Q17 / H19)**:WP §4.4 规则(「低1级:很慢可破坏,可能不掉落」)与示例(「石挖铁:几乎无效」)自相矛盾。本文按「无免疫」裁决统一为:**差1级 = 慢 + 保底掉落**(去掉「可能不掉落」的死锁风险),石挖铁因此可靠破冰、**石→铁 不死锁**。

**方向陷阱(易错点,显式标注)**:
- 战斗:看 **防 − 攻**,差值越大越吃亏(防方占便宜)。
- 采集:看 **工具 − 目标**,差值越小越吃亏(工具不够)。
- 两表减数被减数相反,查表时**先确认「谁减谁」**再读数。白皮书未在同一处提示此差异,实现/讲解时极易写反(见 §5)。

### 2.3 韧性查表(挖掘时间 / 破坏难度 / 抗爆)

**用途边界(WP §5.1)**:韧性 = 对象抵抗破坏、采集、冲击、环境影响的能力。用于 ① 挖掘时间 ② 采集门槛 ③ 方块破坏难度 ④ 爆炸抗性 ⑤ 部分器物强度。**不进武器伤害结算**。

**公式(WP §5.2)**:`最终韧性 = 材料基础韧性 × 形态系数 × 状态系数`

#### 2.3.1 材料基础韧性表(WP §5.2)

| 材料(含骨系/人造同级对齐) | 基础韧性 | 对应等级 |
|---|:---:|:---:|
| 水 | 0 | — |
| 草 | 1 | 木级以下 |
| 土 / 地皮 / 沙 / 苔藓 | 2 | 木级以下 |
| 粘土 | 3 | 木级以下 |
| 木头 | 4 | 木级 |
| 石 / 普通骨 | 5 | 石级 |
| 铁 / 黑骨 | 8 | 铁级 |
| 黄金 / 红骨 | 10 | 黄金级 |
| 黑曜石 / 钢 | 12 | 黄金级(同阶但韧性高于黄金/红骨) |
| 钻石 / 紫骨 | 14 | 钻石级 |

**骨系/人造对齐自洽**:普通骨=石=5、黑骨=铁=8、红骨=黄金=10、紫骨=钻石=14、钢=黑曜石=12。与「骨系装备属性 = 同级矿物」铁律一致(WP §4.2 / §8.1)。
**黄金级内部双韧性档**:黄金/红骨=10,黑曜石/钢=12,二者同为黄金级却韧性不同 —— 说明「等级」与「韧性」是两个独立锁定字段(本文按此理解,需设计层确认,见 §5)。

#### 2.3.2 形态系数表(WP §5.3)

| 形态 | 系数 | 例子 |
|---|:---:|---|
| 松散态 | 0.7 | 沙、碎土 |
| 天然态 | 1.0 | 石、木头 |
| 加工态 | 1.2 | 砖、木板 |
| 装备态 | 1.5 | 剑、甲、采集器 |
| 强化态 | 2.0 | 遗迹墙、黑曜石门 |

#### 2.3.3 状态系数

公式含「× 状态系数」乘项,但**白皮书未给状态系数取值表**(仅出现在公式中,无清单)。查表时**默认状态系数 = 1.0(占位),待后续设计补值**;凡涉及状态系数的算例均显式按 1.0 计并标注。

### 2.4 四个结算示例(逐步可复现)

> 约定:阶级索引 木=0/石=1/铁=2/黄金=3/钻石=4;黑曜石=黄金级=3。

**示例 A — 木剑打铁级怪(战斗,WP §4.3 原文例)**
攻=木剑=0;防=铁级怪=2。差值 = 防 − 攻 = 2 − 0 = **2** → 查压制表 → **15% 伤害**。
结论:木剑能打铁怪,但只有 15%(刮痧),非零、可磨 —— 印证「低级可屠龙,只是极慢」。

**示例 B — 铁剑打黄金级怪(战斗,WP §4.3 原文例)**
攻=铁剑=2;防=黄金级怪=3。差值 = 防 − 攻 = 3 − 2 = **1** → 查压制表 → **45% 伤害**。
结论:差 1 级仍有近半伤害,黄金怪对铁装玩家「硬但可打」,符合「正式装备阶」定位。

**示例 C — 黄金级采集器采黑曜石(采集,WP §4.4 原文例)**
工具=黄金级采集器=3;目标=黑曜石=黄金级=3。差值 = 工具 − 目标 = 3 − 3 = **0** → 查采集表 → **正常采集、正常掉落**。
延伸:钻石级采集器(4)采黑曜石(3),差 +1 → 仍「正常/更稳」(WP §4.4「更稳」无量化);石级采集器(1)采黑曜石(3),差 −2 → **极慢 + 减产但仍有产出**(无免疫,非硬墙;惩罚性低效,没人会当主路)。

**示例 D — 黑曜石门的最终韧性(韧性,WP §5.2/§5.3)**
材料=黑曜石,基础韧性=**12**;形态=强化态(遗迹墙/黑曜石门),系数=**2.0**;状态系数按占位 **1.0**。
最终韧性 = 12 × 2.0 × 1.0 = **24.0**(状态系数待补,届时按比例缩放)。
对比:天然石(5 × 1.0 × 1.0)=5.0;黑曜石门约为天然石的 **4.8 倍**,符合「高韧性特殊建筑」定位。

### 2.5 三套规则的自洽结论

1. **非零下限 = 房主拍板确认**:战斗表底 1% 而非 0%,允许低级屠龙(「连钻石龙站着也能杀」)—— 锁定模型刻意设计,**不计为漏洞**。
2. **采集亦「无免疫」= 房主拍板**:撤销先前「采集封墙」的非对称——战斗与采集同遵「只减不免」:差1级=慢+保底掉落,差≥2=极慢减产但永不归零。防跳级靠惩罚性低效,而非硬墙(已决 Q17,见 §2.2)。
3. **韧性不串武器伤害 = 确认**:WP §5.1 显式排除,三套规则解耦自洽。
4. **骨系/人造韧性对齐 = 确认**:普通骨/黑骨/红骨/紫骨/钢的韧性与同级矿物一致,不破坏「骨系=同级矿物」铁律。

## 3. 掉落 / 食物 / 附性归一化

> 沉淀「生物给玩家什么、食物怎么恢复、显现能附带什么」三件事的归一化模型。三者共用同一条「原型锁定·显现自由」铁律:逻辑只认原型字段(掉落池 / 恢复锚点 / 阶级),显现(贴图 / 名字 / 附性)再花也改不了结算(造物 §2)。

### 3.1 一条铁律管三张表

| 维度 | 原型锁定(本质,逻辑结算用) | 显现自由(表象,玩家可见) |
|---|---|---|
| 食物 | 「苹果型食物」原型 → 基础恢复锚点(WP §3.4) | 显现名 / 贴图 / 分类 / 来源 / 轻微附性(WP §3.4) |
| 掉落 | 生物等级 → 掉落池(食物 / 骨 / 油 / 经验球 + 高级额外)(WP §3.5、§10.1) | 食物显现为何种果、骨显现为何种骨爪材料(WP §3.5) |
| 附性 | 不可改变原型本质;不可跨阶级(WP §2.3) | 少量回魔 / 微弱发光 / 短暂抗寒 / 轻微中毒(WP §2.3、§3.4) |

**铁律(WP §2.2、造物 §2)**:钻石色小鸡若为木级猎物,就不能掉钻石;名为「神兽」的低阶生物不能有 Boss 属性。

### 3.2 食物归一化:一个原型管所有食物

#### 3.2.1 原型定义(WP §3.4)
食物本质统一为**「苹果型食物」原型**(「苹果」是食物原型名,非具体水果)。
- 食物原型决定**基础恢复价值**(本文称「恢复锚点」)。
- 不同显现可拥有:贴图、分类、来源、**轻微附加属性**。
- **白皮书未给食物恢复的具体数值**(连标准恢复=多少点都未给)。本节只能给结构锚点,数值待立桩(见 §5)。

#### 3.2.2 食物显现枚举(WP §3.4)
可显现为:苹果、蓝莓、蘑菇、藤蔓果、蜂蜜果、肉、烤肉、发光果、辛辣果。

#### 3.2.3 食物显现表(恢复锚点 + 来源 + 附性)
> 「恢复锚点」列以「标准恢复」为唯一基准锚(WP §3.4 把苹果标为「标准恢复」),其余显现相对标准锚标注;**所有具体点数白皮书未给**,以定性锚 + 待立桩占位表示,禁止臆造数字。

| 显现 | 恢复锚点(相对「标准恢复」) | 来源 | 附性 | 出处 |
|---|---|---|---|---|
| 苹果 / 红苹果 | 标准恢复(基准锚) | 树 / 植物 | 无 | WP §3.4、§2.3 |
| 蓝莓 | 标准恢复(未给差异) | 森林灌木 | 少量恢复魔力 | WP §3.4、§2.3 |
| (发光)蘑菇 | 标准恢复(未给差异) | 洞穴 | 短时间微弱发光 | WP §3.4、§2.3 |
| 藤蔓果 | 标准恢复(未给差异) | 植物(枚举,来源细化未给) | 未给 | WP §3.4 |
| 蜂蜜果 | 标准恢复(未给差异) | 未给 | 未给 | WP §3.4 |
| 发光果 | 标准恢复(未给差异) | 未给 | 微弱发光(同发光蘑菇族,未单列) | WP §3.4 |
| 肉 | 标准动物食物(标准恢复) | 生物掉落 | 无(标准) | WP §3.4、§3.5 |
| 烤肉 | 标准恢复(未给差异;通常熟食 > 生肉,未给) | 肉加工(配方未给) | 未给 | WP §3.4 |
| 辛辣果 | 标准恢复(未给差异) | 火山植物 | 短暂抗寒 | WP §3.4、§2.3 |
| 毒浆果 | 「恢复生命但轻微中毒」(净恢复未给) | 来源未给 | 恢复生命 + 轻微中毒 | WP §2.3 |

**附性边界(WP §2.3)**:无论显现为蓝莓还是毒浆果,结算时都按「苹果型食物」原型走恢复锚点,附性只是叠加的小标签 —— 食物可有小效果,但不能变成钻石资源。

#### 3.2.4 恢复锚点的三段锚(供后续立桩,不含数值)
1. **基准恢复值** = 「标准恢复」(苹果 / 肉),所有食物以此为 1.0 锚。
2. **显现微调** = 同原型不同显现的轻微差异(WP §2.3 允许「小效果」,但未给食物间恢复差异是否存在;**默认所有食物同恢复锚,差异只走附性** —— 这是 §5 的关键开放问题)。
3. **附性效果** = 与恢复值正交的副作用(回魔 / 发光 / 抗寒 / 中毒),不计入恢复锚。

### 3.3 掉落归一化:生物等级 → 掉落池

#### 3.3.1 通用掉落(WP §3.5)
普通生物主要掉四类,按原型意义归一:

| 掉落 | 原型意义 | 出处 |
|---|---|---|
| 食物 | 苹果型食物的**动物显现,常表现为肉** | WP §3.5 |
| 骨 | 生物硬质材料,**按等级分阶**(普通骨 / 黑骨 / 红骨 / 紫骨) | WP §3.5、§4.2 |
| 油 | 燃烧、照明、燃烧瓶、机关、熔炉相关资源 | WP §3.5 |
| 经验球 | 角色成长资源(吸收升级,每级 +1 属性点) | WP §3.5、§9.1 |

#### 3.3.2 高级怪额外掉落(WP §3.5)
高级怪在通用掉落之外可额外掉:**铁、黄金、黑曜石、钻石、技能书、成品物品**。

#### 3.3.3 等级 × 掉落映射(WP §10.1,与骨系等级 §1.3 对齐)

| 生物等级 | 生物定位 | 主要掉落 | 骨显现 | 出处 |
|---|---|---|---|---|
| 木级 | 猎物、小动物 | 食物、油、少量经验 | **无骨掉落**(无木级骨) | WP §10.1、§4.2 |
| 石级 | 普通怪 | 普通骨、食物、油、经验 | 普通骨(石级) | WP §10.1、§4.2 |
| 铁级 | 掠食者、强怪 | 黑骨、食物、油、经验 | 黑骨(铁级) | WP §10.1、§4.2 |
| 黄金级 | 精英、守卫 | 红骨、油、经验、黄金 / 黑曜石 / 成品 | 红骨(黄金级) | WP §10.1、§4.2 |
| 钻石级 | Boss | 紫骨、钻石、技能书、高阶成品 | 紫骨(钻石级) | WP §10.1、§4.2 |

**关键对齐:木级无骨掉落** —— 掉落池与「无木级骨头」(WP §4.2)严格一致,木级猎物只产食物 / 油 / 少量经验。

#### 3.3.4 掉落数值现状(WP §3.5)
掉落的概率 / 数量,白皮书只给定性(「主要掉」「可额外掉」),**未给任何具体数值**。故掉落表只锁**掉落池构成**与**等级映射**,概率与数量待立桩(见 §5)。

#### 3.3.5 与生态位的衔接(WP §10.2)
掉落的「拿到难度」由生态位调制:猎物(逃跑)= 食物来源,易取;守卫 = 资源门槛,守护遗迹 / 宝箱;Boss = 阶级突破,掉技能书 / 高阶成品。**生态位不改掉落池,只改获取方式**(WP §10.2、§11.1「群落只改显现与分布,不增新材料」同源)。

### 3.4 附性归一化:正交的轻微标签

#### 3.4.1 附性定义与示例(WP §2.3、§3.4)
显现可带**轻微附加属性**,称「附性」—— 挂在显现上、与原型本质正交的小标签。

| 显现 | 原型 | 附性 | 出处 |
|---|---|---|---|
| 红苹果 | 食物 | 无 | WP §2.3 |
| 蓝莓 | 食物 | 少量回魔 | WP §2.3、§3.4 |
| 发光蘑菇 | 食物 | 短时间微弱发光 | WP §2.3、§3.4 |
| 辛辣果 | 食物 | 短时间抗寒 | WP §2.3、§3.4 |
| 毒浆果 | 食物 | 恢复生命但轻微中毒 | WP §2.3 |

#### 3.4.2 附性硬边界(WP §2.3)
附性**不能改变原型本质**,三条不可逾越:
1. 食物可有小效果,但**不能变成钻石资源**(不能靠附性跨阶产出稀有资源)。
2. 低级怪可跑得快一点,但**不能拥有钻石级防御**(附性不跨阶级 buff)。
3. 同阶武器可略快或略远,但**不能跨阶级**(对齐 WP §7.2 剑显现偏置 ±5%、§8.1 真实属性=槽位×等级)。

**结论**:附性 = 显现层的「风味调料」,永远在原型阶级天花板之下。逻辑结算只读原型字段,附性只触发副作用(回魔 / 发光 / 抗寒 / 中毒计时),不参与阶级、伤害、掉落池的核心结算。

#### 3.4.3 附性数值现状(WP §2.3、§3.4)
附性的具体量(回多少魔、发光半径 / 时长、抗寒时长 / 减伤、中毒 DPS / 时长)白皮书**全部未给**,只给定性「少量 / 短时间 / 轻微」。待立桩(见 §5)。

## 4. 经济闭环与进度门(对抗性审查)

> 方法:先把三条路线(采集 / 狩猎 / 制造)各自能否触达五阶级走一遍,标注冗余 / 断档 / 不可达;再把游玩层三门(工作台材质阶级 / 图纸 / 材料)映射到五阶级,产出「每升一阶需解锁什么台 + 什么图纸 + 什么料」的进度门表。**凡白皮书未给数字处照实标注「未给」,不补**。

### 4.1 三路线 × 五阶级触达矩阵

**符号**:✓=有明确来源;⚠=有来源但门槛/断档存疑;✕=该路线在此阶级无材料产出(断档)。

| 阶级 | 采集路线(挖矿/伐木) | 狩猎路线(骨系) | 制造路线(人造材料) | 触达结论 |
|---|---|---|---|---|
| **木级** | ✓ 木头(平原/森林,WP §11.2) | ✕ **无木级骨**(WP §4.2 明文) | — 开局无人造材料 | 仅采集路线可达 |
| **石级** | ✓ 石(平原/森林/山地) | ✓ 普通骨(森林/石级怪,WP §10.1) | — | 采集+狩猎双通 |
| **铁级** | ✓ 铁(山地) | ✓ 黑骨(湿地/山地铁级怪) | — | 采集+狩猎双通 |
| **黄金级** | ✓ 黄金/黑曜石(火山) | ✓ 红骨(火山黄金级怪) | ✓ **钢**=铁+大量低料+技能书+工艺(WP §12.3) | **三路线全通(唯一)** |
| **钻石级** | ✓ 钻石(深层遗迹) | ✓ 紫骨(钻石级 Boss) | ✕ **制造断档**:钢封顶黄金、不可递归升钻石(WP §12.3) | 采集+狩猎可达,制造不可达 |

**两处结构性断档(均为白皮书有意设计,但需在进度门显式承接)**:
- **木级狩猎断档**:WP §4.2 明文「没有木级骨头」,狩猎路线从石级起步。木级阶段狩猎只产**食物/油/少量经验**(WP §10.1),不产可升阶硬质材料。新手第一阶「变强」完全依赖采集路线(木头)。
- **钻石级制造断档**:WP §12.3「钢不能继续升级成钻石级(封顶不递归)」。终局阶**没有「造」出来的路**,钻石级硬质材料只能靠采集(钻石)或狩猎(紫骨)。制造路线最高只到黄金级。

### 4.2 冗余检查:钢 vs 黄金矿 是否各有不可替代理由

WP §12.3 明文「两条黄金级路线:探索路线(黄金、红骨、黑曜石)、制造路线(铁→钢)」。对抗性问:健康冗余还是死冗余?**已由房主拍板的「工作台加工规则」彻底分开两路,确认健康冗余。**

> **★ 工作台加工规则(房主拍板,2026-06-13)**:一座工作台**只能加工 ≤ 自身阶级的天然材料 + 图纸解锁的特定人造材料**(其成品阶级**可高于**台阶级,如钢);**不能加工高于自身阶级的天然材料**。
> 推论:**铁级台(或黑骨同级台)+ 高级图纸 → 钢(黄金级成品)** —— 这正是「人造材料可比主原料高一阶」(WP §12.3)的落地点;但铁级台**碰不了黄金/黑曜石/钻石等天然高阶材料** —— 要加工天然黄金/黑曜石,须黄金级台(其升级料黄金/黑曜石仅产火山,故**天然黄金线受火山门把守**)。
> 这条规则即 Q24「工作台阶级 vs 材料阶级谁决定可造性」的**通用答案**:**人造材料看图纸(可越台阶级),天然材料看台阶级(不可越级)。**

| 维度 | 探索/采集(黄金矿、黑曜石) | 制造(铁→钢) | 区分(已决) |
|---|---|---|---|
| 群落门槛 | 必须打进**火山** | **铁级台 + 高级图纸**,不进火山 | ✓ **钢是真旁路**:铁台造钢成立(成品=黄金级、加工台=铁级),不碰火山 |
| 前置投入 | 黄金级采集器 + 抗火山怪 | 铁 + 大量低料 + **高级图纸** | ✓ 钢用「低料堆量 + 高级图纸」换「高危探索」 |
| 阶级天花板 | 可达**钻石**(钻石/紫骨) | **封顶黄金**(无人造钻石) | ✓ 各有上限差:制造流是黄金级捷径但到顶,天然流更险但通终局 |
| 韧性数值 | 黄金=10、黑曜石=12 | 钢=12(等同黑曜石) | ⚠ 钢/黑曜石同韧性同阶,需靠成本拉开(Q19) |

**冗余结论(健康,已决 Q24)**:钢的不可替代理由 =「**不进火山的黄金级捷径,但封顶黄金**」;天然流的不可替代理由 =「**更危险,但唯一通钻石**」。两路各有进出口与天花板,确认健康冗余。
**残留专项**:钢(韧性12)与黑曜石(韧性12)同阶同韧性,须靠钢的「大量低级材料」成本与黑曜石稀缺度联合定标,防成本架空(Q19,入 §5)。

### 4.3 弓 / 法杖的材料出口是否映射到阶级路线

§4.1 矩阵只覆盖「硬质材料」。三武器原型(WP §7)各有不同资源出口,需单独核对:

| 武器 | 消耗 | 消耗品来源 | 阶级映射是否清晰 |
|---|---|---|---|
| 剑 | 无消耗(WP §7.1) | — | ✓ 纯靠武器本体阶级 |
| 弓 | **箭**(WP §7.3) | 箭由木头/石/骨制造(WP §7.3) | ⚠ **箭的阶级与弓的阶级关系白皮书未给**:石箭配钻石弓打钻石怪,按攻击阶级取谁? |

### 4.4 游玩层三门 × 五阶级 进度门表(核心交付)

| 升入阶级 | ① 工作台(基础设施) | ② 图纸/技能书(任务/探险) | ③ 材料(决定成品阶级) | 群落/来源(WP §11.2) | 该阶可造(WP §12.2) |

**三门齐备的对抗性自检(逐阶)**:
- **木→石**:台✓、图纸⚠(基础图纸是否开局自带还是需第一个任务?WP未给)、料✓。**风险**:若石阶图纸需「探险获得」而探险又需石阶装备,产生**鸡生蛋循环** —— 必须保证石阶基础图纸开局/低门槛可得(入 §5)。
- **石→铁**:台✓、图纸✓、料✓(铁在山地)。**已决 Q17**:房主拍板「无免疫」,差1级=慢+**保底掉落**,石镐可靠挖到铁、**自举不死锁**;仅「慢」的节奏数值待立桩(非阻断)。
- **铁→黄金**:台✓、图纸✓(钢技能书)、料✓(三入口最丰富)。**唯一三门三料全通的阶级**,设计余量最大。
- **黄金→钻石**:台✓、图纸⚠(钻石图纸只在 Boss/深层遗迹掉)、料✓(但制造断档)。**风险**:进深层遗迹需黄金/钻石级装备(WP §11.2),钻石装备图纸又在遗迹里 → 又一处**循环依赖**,需黄金级装备能「低效屠龙」破冰(WP §4.3 非零下限 1% 支持,但效率存疑;入 §5)。

### 4.6 闭环完整性总评

- **采集路线**:五阶级全通(木→石→铁→黄金→钻石),唯一无断档的主干路线。
- **狩猎路线**:石→钻石四阶全通,**木级缺位**(无木级骨,有意设计)—— 新手第一阶不能纯靠打怪升阶硬质。
- **制造路线**:仅黄金级一阶有产出(钢),**木/石/铁/钻石四阶无人造材料升阶** —— 制造路线是「单点旁路」而非「平行主干」,且终局(钻石)制造断档。
- **冗余健康(已决 Q24)**:黄金级三路线汇聚是设计高潮;经「工作台加工规则」拍板,钢 = 铁级台 + 高级图纸 → 黄金级成品,**确认真旁路**(不进火山、封顶黄金),与「更险但唯一通钻石」的天然流构成健康冗余。残留:钢/黑曜石同韧性需靠成本拉开(Q19)。
- **系统性风险已大幅缓解(无免疫裁决)**:先前的「采集器自举死锁链」因「无免疫」拍板而解除——差1级慢但**保底掉落**、战斗差≥4 仍 1%,各处不再是硬死锁,只剩「慢/低效」的**节奏数值待立桩**(非阻断,黄金→钻石的低效屠龙效率仍需实测,见 Q18)。
