# 场景 01:本机 / 局域网开黑

## 目标

在一台 Windows 电脑上把服务器跑起来,本机浏览器先进游戏;
同一 Wi-Fi 下的手机(安卓 / iPhone 浏览器均可)直接扫地址加入,全程不需要外网。

## 前提

- 已安装 Node.js(架构契约按 Node.js 24 编写,见 `DESIGN.md`)。
- 仓库已克隆到本地,并执行过一次 `npm install`(中继运行时只依赖 `ws` 一个外部包)。
- 手机和电脑连**同一个 Wi-Fi**(同一路由器、同一网段)。

## 操作步骤

1. 启动服务器,二选一:
   - **命令行**:在仓库根目录执行 `npm start`(等价于 `node server/index.js`)。
   - **双击 `start.cmd`**:它会切到脚本所在目录启动服务器,并在约 2 秒后自动用默认
     浏览器打开 `http://localhost:8080`;关掉那个黑色服务器窗口即停服。
2. 看控制台横幅。启动成功后会打印所有可访问地址,形如:

   ```
   ==============================================
     VoxelCraft server is running
   ----------------------------------------------
     http://localhost:8080
     http://192.168.x.x:8080
   ----------------------------------------------
     手机需连接同一 Wi-Fi，访问上面的局域网地址。
     (Phones must be on the same Wi-Fi network.)
   ==============================================
   ```

   横幅会列出本机每一个非回环 IPv4 地址(`server/index.js` 枚举网卡生成)。
3. 第一次启动时 Windows 防火墙可能弹窗询问是否允许 Node.js 联网——
   **必须勾选"专用网络"并允许**,否则手机连不上。
4. 电脑本机:浏览器打开 `http://localhost:8080`,建人物 → 进大厅 →
   输入房间名点"进入房间"(没有就会提示创建)。
5. 手机:浏览器访问横幅里的局域网地址(如 `http://192.168.x.x:8080`),
   建人物后在大厅房间列表里点同一个房间即可加入。
   手机网页里"服务器地址"一栏**留空**即可——页面本身就是这台服务器发出来的,
   留空表示连本站。

## 预期结果

- 服务器控制台每个生命周期事件打一行日志:房间创建/删除、成员进出、房主迁移。
- 同房间内所有人实时互见(含人物配色),方块修改全员同步,后进的人也能看到此前全部改动。
- 一个房间最多 8 人(房主 + 7 名成员)。

## 常见故障与排查

| 现象 | 原因与处理 |
|---|---|
| 启动即报"端口 8080 已被占用 (port in use)" | 已有一个实例在跑(或端口被别的程序占了)。按提示换端口:`$env:PORT=8081; npm start`(PowerShell);换端口后手机访问地址里的端口也要跟着改。 |
| 电脑能玩,手机打不开页面 | ① 防火墙没放行 Node.js:到"Windows 安全中心 → 防火墙 → 允许应用通过防火墙"勾上 Node.js 的专用网络;② 手机连的不是同一 Wi-Fi(比如走了流量或连了访客网络);③ 路由器开了 **AP 隔离 / 客户端隔离**,设备间互 ping 不通——关掉该选项或换路由器;④ 访问的 IP 抄错了,以横幅打印为准。 |
| 横幅列出多个局域网地址,不知道用哪个 | 逐个试,通常 `192.168.` 开头的那个是家用路由器网段;虚拟网卡(虚拟机、VPN)的地址手机不可达。 |
| 手机页面打开了,但点"刷新"显示未连接 | 页面是静态文件,WebSocket 才是连接本体。确认地址栏是 `http://`(不是 `https://`)——局域网裸 IP 没有证书,`https` 页面会因混合内容拦掉 `ws://`。直接用横幅里的原始地址即可。 |
| 进房后另一台设备看不到我 | 看服务器控制台有没有 `peer #x joined` 日志;没有则两台设备连的可能不是同一个服务器实例(比如一台连了 8080、一台连了 8081)。 |
| 想停止服务器 | 直接关服务器窗口或按 `Ctrl+C`。房主标签页才是房间权威,中继一停所有连接断开,所有房间消失(无存档)。 |
